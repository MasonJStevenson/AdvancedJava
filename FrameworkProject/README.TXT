AUTHORS: Mason Stevenson

PURPOSE OF PROJECT: The goal of this project is to create an "application framework" for some problem domain.

HOW TO START THIS PROJECT: To run this program, simply run the main main method of the CalcDriver class in 
the default directory. The blue box in the upper-righthand corner contains a list of all currently available plugins.
To load a plugin, click its name in the blue layout selection box.
